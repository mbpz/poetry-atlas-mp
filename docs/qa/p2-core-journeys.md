# P2 核心旅程真机验收

适用版本：`2026-q3-batch-1`。每轮至少覆盖 1 台 Android 真机、1 台 iPhone，并分别在正常网络与弱网/断网下执行。

## 地图 → 地点 → 诗文

1. 首次进入地图，完成引导；切换到“唐”，拖动并缩放地图。
2. 点击地点，确认热榜预览被完整诗文替换后不出现重复卡片。
3. 加载更多后进入一首诗，再连续返回地点和地图。
4. 预期：地点筛选、列表位置和已加载数量保留；地图朝代、中心和缩放保留。
5. 断网重复步骤 1–2，确认地图和地点页出现可见错误与重试；地图旧标记仍可点击。恢复网络后重试成功。

## 搜索 → 阅读 → 收藏

1. 搜索“西湖”，切换“诗词/作者/地点”，滚动后进入诗词详情。
2. 确认详情展示全文/节选状态、文本来源、许可或“来源待补录”，作者和地点可进入。
3. 快速连续点击收藏，预期只执行一次；进入收藏页后状态一致。
4. 返回搜索，预期关键词、Tab 和滚动上下文保留。
5. 断网搜索和收藏，确认出现内联错误；收藏失败恢复原状态，恢复网络后可重试。

## 我的 → 收藏 / 路线

1. 修改昵称、头像；正常网络下保存成功，重新进入仍一致。
2. 断网修改昵称、头像；预期恢复旧值并显示失败原因。
3. 创建路线：验证空名称、空站点、只填一个坐标、越界坐标均不能提交，且输入不丢失。
4. 正常保存后快速重复点击，预期只生成一条路线；个人路线统计同步。
5. 在路线列表使用删除按钮，取消确认不删除；断网删除保留原列表并可重试；正常删除后统计同步。
6. 使用另一个微信账号验证：无法看到或打开前一账号的收藏、路线和用户资料。
